# Angular 4 Todo App.

---

## What it is
Just another todo app I started to develop from scratch. This app supports the following features in the current version.


### Features
* Filter states
* Add / Remove / Rename function
* Detail view
* Counter
* Navigation
* LocalStorage

```npm install```
